# PythonProject2

爬取阿里公益，腾讯公益，微博公益等的项目

## 功能特性
- ali.py,alis.py是爬取阿里公益网的项目并存储到Excel中;
- fe.py,yi.py是爬取腾讯公益项目并存储到Excel中;
- xin.py,xins.py是爬取微博微公益项目并存储到Excel中;
- diao.py是自动化填写问卷星的调查问卷;
- jie.py,main.py是在易盾网利用超级鹰在线体验文字点选功能的;
- ur.py是随机UA设置


## 快速开始
​```bash
git clone https://github.com/llk-1212/PythonProject2.git
cd PythonProject2
npm install
npm start
​```
